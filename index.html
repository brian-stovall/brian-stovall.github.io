Selected Work

Maze Explorer

I have always loved mazes. I was reading about maze generating algorithms one day and started experimenting, and this is the result (so far). The program builds a maze and then lets the user explore in different modes, based on what gear they decide to bring along. Use vim or arrow keys to explore.

Melanaza Timer

I built a pomodoro clock web app, but I like eggplants, so I used an eggplant rather than a tomato for the graphic. It animates via a div element with a higher z-index "growing" and "shrinking" over a static eggplant image instead of a canvas or DOM sprites.

Fruit-Up

When I first started to make webpages, I was suprised how much processing power it seemed to take to make simple animations compared to native applicatons. I read about using DOM elements as sprites instead of canvas or libraries like Phaser, and I made this silly "screensaver" webpage to stress-test the idea a bit and try out DOM spritesheets.

Simon

A webpage rendition of the 'Simon' game. This was early practice in shaping divs, controlling state in a webpage, and handling events, sound, and animation in the browser.
